# 🖼️ Image Gallery

A responsive and visually appealing Image Gallery built using **HTML, CSS, and JavaScript**. The project displays images in an organized layout with a clean and user-friendly interface.

## 🚀 Features

- Responsive image gallery layout
- Multiple image categories
- Clean and modern UI
- Smooth hover effects
- Easy navigation
- Lightweight and fast

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Image-Gallery/
│── index.html
│── style.css
│── script.js
│── images/
│── README.md
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your browser.

## 📸 Preview

The gallery displays different image categories in a responsive layout with an attractive design.
